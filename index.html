Tarefa 1

Descrição da Tarefa

O objetivo desta tarefa era fazer cada um dos processos testar o seguinte no anel

Arquivos da tarefa

  • tempo1.c
  • log 0
  • log 1
  • log 2

Relatos da tarefa

Utilizando como base o código da tarefa 0, houve a necessidade de imprementar uma variável int que aloque o id do próximo processo (chamada de proximo). O id do próximo processo de N-1 foi tratado para que seja o 0 (zero), para isso, foi usado proximo = (token + 1) % N. Foi criada a variável situacao que recebe um binário que representa o status do processo (0-> Correto e 1-> Falho). Testes removendos diferentes nós podem ser vistos nos logs
